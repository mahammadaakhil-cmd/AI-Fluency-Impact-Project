# AI Fluency Impact Project – Personal Portfolio Website

## 📌 Project Overview

This project is a responsive personal portfolio website created as part of the **General AI Fluency – Impact Project**. It showcases my profile, technical skills, projects, internships, certifications, and contact information in a clean and modern design.

The website serves as my professional portfolio and can be used by recruiters, mentors, and collaborators to learn more about my work.

---

## 🚀 Features

- Responsive design for desktop and mobile
- About Me section
- Technical Skills
- Projects Showcase
- Internship Experience
- Certifications
- Contact Information
- Resume Download
- Smooth scrolling navigation
- Section fade-in animations
- Interactive navigation highlighting
